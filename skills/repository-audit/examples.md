# Examples

## Basic read-only audit

```text
Use the Repository Audit Skill. Perform a full read-only audit of this repository. Do not modify source code. Do not run build, test, analyze, lint, install, or formatter commands. Create the audit report in Russian and save it to audit_reports/YYYY-MM-DD-audit-report.md.
```

## Flutter repository audit

```text
Use the Repository Audit Skill. Audit this Flutter repository in read-only mode. Pay special attention to lib/, pubspec.yaml, platform folders, state management, navigation, data persistence, permissions, generated files and tests. Save the Russian report to audit_reports/YYYY-MM-DD-audit-report.md.
```
